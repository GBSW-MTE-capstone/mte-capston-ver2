## 프론트 뚝딱중입니당

`git clone https://github.com/GBSW-MTE-capstone/mte-capstone-ver2`
이렇게 cmd 에서 클론하고

`cd mte-capstone-ver2`
폴더로 들어가서 

`npm i`
로 node module 을 다운로드 한후 

`yarn`
모듈 다운 하시고 

`yarn dev`
로 실행시키셈요.

더러운 코드로 인한 안구피해는 책임지지 않습니다.
